Creating a shortcode to display Postnord Tracking information to end-user.
Connects to the public API of Postnord in order to track shipments made on Postnord

== Description ==
This plugin connects to  the API of Postnord  and creates a shortcode that can be inserted anywhere in your woocommerce site.
It creates a form where you can search based on shipment ID.
Place shortcode `[woo-postnord-tracking-form][/woo-postnord-tracking-form]` in any page or post to render the form.
In order to use this plugin you will need an API that can be requested for free from [Postnord Developer Portal](https://developer.postnord.com/)
